pivot_table vs. groupby¶

see https://stackoverflow.com/questions/34702815/pandas-group-by-and-pivot-table-difference

df = pd.DataFrame({"a": [1,2,3,1,2,3], "b":[1,1,1,2,2,2], "c":np.random.rand(6)})
df
a b c
0 1 1 0.122273
1 2 1 0.785140
2 3 1 0.003996
3 1 2 0.669116
4 2 2 0.599575
5 3 2 0.095478

A pivot table creates a new dataframe, with the specified column in the original treated as the index, and any columns given as columns in the dataframe, with the values taken to be “c”

df.pivot_table("c", index=["a"], columns=["b"])
b 1 2
a
1 0.122273 0.669116
2 0.785140 0.599575
3 0.003996 0.095478
